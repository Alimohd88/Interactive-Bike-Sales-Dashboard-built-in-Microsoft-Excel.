# 🚴 End-to-End Bike Sales Analysis in Microsoft Excel

## 📌 Project Overview

This project demonstrates a complete data analysis workflow using Microsoft Excel, starting from raw data and ending with an interactive dashboard.

The objective was to clean, transform, analyze, and visualize the Bike Sales dataset to uncover customer purchasing patterns and business insights.

---

## 🔄 Project Workflow

### 📥 Data Collection

* Imported the Bike Sales dataset into Excel.

### 🧹 Data Cleaning

* Removed inconsistencies
* Standardized categorical values
* Handled missing or duplicate data where required
* Created Age Brackets

### 🔧 Data Manipulation

* Created calculated columns
* Organized data for reporting
* Prepared data for analysis

### 📊 Exploratory Data Analysis (EDA)

* Customer demographics
* Income analysis
* Occupation analysis
* Region analysis
* Education analysis
* Commute distance analysis
* Bike purchase trends
